# Hack The Shields 2026 - Write-ups

Ce dépôt contient l'ensemble des write-ups (solutions détaillées) pour les challenges du CTF **Hack The Shields (HTS) 2026**.

Chaque solution détaille le cheminement technique, les outils utilisés et le flag final permettant de valider le challenge.

## 📁 Structure du Dépôt

Les solutions sont organisées par catégorie de challenge, puis par niveau de difficulté :

```text
📂 <Catégorie>
 ┗ 📂 <Difficulté> (Facile / Intermédiaire / Difficile)
   ┗ 📂 <Nom du Challenge>
     ┗ 📝 write-up.md
```
